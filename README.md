# QA API Testing Project

Projeto de testes de API REST desenvolvido com Postman para demonstrar conhecimentos em validação de endpoints, autenticação e testes funcionais.

## Objetivos

Validar o comportamento da API através de diferentes cenários positivos e negativos.

## Cenários Testados

- GET - Consulta de produtos
- POST - Criação de registros
- PUT - Atualização de registros
- DELETE - Exclusão de registros
- Autenticação
- Testes negativos
- Validação de status code
- Validação de resposta JSON

## Tecnologias Utilizadas

- Postman
- REST API
- JSON
